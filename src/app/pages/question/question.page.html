Sobre o TecAuton

Plataforma para técnicos de
celulares autônomos.

Este app foi criado para o auxílio
de técnicos, que necessitam manter
controle de suas demandas de serviços.

Cadastro de usuário:

Cadastrar usuário para ter acesso a
plataforma.

Cadastro de cliente:

Cadastrar os dados do cliente para
controle das Ordens de Serviços.

Cadastro de OS:

Cadastrar os dados dos aparelhos, para
gerar as Ordens de Serviços.

Fluxo de Caixa:

Registrar saída de caixa, gerar relatório
de serviços, relatório de saída de caixa
e impressão de recibo.
